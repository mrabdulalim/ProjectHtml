Cyber Security





How to keep systems secure?

  • Update security software regularly
  • Log out when not using a computer
  • Do not download from unverified sources
  • Back up data
  • Encrypt data
  • Do not post sensitive information on social media
  • Be aware of your surroundings
  • Do not click on suspicious emails
  • Install anti virus software
  • Use a secure Wi-Fi connection
  • Limit access to those who require it
  • Delete data that is no longer useful
